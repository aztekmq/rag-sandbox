# rag-sandbox
A single Docker container with a beautiful WebUI offering:  Admin mode → Upload new IBM MQ PDFs, re-index, delete documents, view logs, manage system prompt User mode → Simple chat interface (no upload rights) Runs fully offline with Llama-3.1-8B-Instruct Q5_K_M GGUF (best quality/speed trade-off in 2025) Zero external API calls, full data privacy
